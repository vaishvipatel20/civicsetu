# 🏙️ Smart Civic Issue Reporting and Management System

*A Smart India Hackathon project developed to streamline civic issue reporting through a cross-platform mobile application built with React Native and Expo.*

---

📖 Project Overview

The Smart Civic Issue Reporting and Management System is a cross-platform mobile application developed as part of the Smart India Hackathon (SIH) to simplify the reporting and management of civic issues. The application enables citizens to report problems such as potholes, waste accumulation, damaged streetlights, and water leakage by submitting detailed complaints with photographs and GPS location.

Built using React Native, Expo, and TypeScript, the application integrates location services, image capture, multilingual support, interactive maps, and a management dashboard to provide an efficient, user-friendly complaint management platform. The system promotes transparent communication between citizens and municipal authorities while enabling faster issue tracking, prioritization, and resolution through a centralized digital interface.

---

🎯 Project Objective

The primary objective of this project is to develop a centralized platform that allows citizens to report civic issues quickly while enabling authorities to efficiently monitor, prioritize, and manage complaints. By integrating real-time location tracking, image-based reporting, and an intuitive mobile interface, the system aims to improve response time, strengthen communication between citizens and government agencies, and support smarter decision-making through organized data management.

---

💡 Motivation

Traditional complaint management systems are often time-consuming, fragmented, and difficult to track, leading to delayed resolution of public issues. This project was developed to address these challenges by providing a modern, technology-driven platform that simplifies issue reporting and improves communication between citizens and local authorities. It also gave me practical experience building scalable mobile applications using modern frameworks, geolocation services, and user-centric design principles.

---

# 🛠️ Technology Stack

The application was developed using modern mobile development technologies to ensure scalability, performance, and a seamless user experience across Android and iOS platforms.

- **React Native** – Cross-platform mobile application development.
- **Expo** – Simplifies development, testing, and deployment.
- **TypeScript** – Provides type safety and improves code maintainability.
- **Expo Router** – File-based navigation and routing.
- **React Navigation** – Enables smooth navigation between application screens.
- **Expo Location** – Retrieves real-time user location for issue reporting.
- **Expo Image Picker & Camera** – Captures and uploads images as complaint evidence.
- **i18next** – Supports multilingual user interfaces.
- **React Native Maps** – Displays issue locations and map-based visualization.
- **Async Storage** – Stores local application data and user preferences.

---

# ⚙️ Working Principle

Users can register complaints by selecting the issue category, providing a description, attaching photographs, and sharing their current location. The application validates the submitted information and records the complaint for further processing. Municipal authorities can access submitted reports, review complaint details, assign priorities, monitor progress, and update the resolution status through the administrative interface.
